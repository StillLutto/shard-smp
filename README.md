> The Official Plugin for the Shard SMP!

## Install

Run this to generate a jar file:
```sh

./gradlew  reobfJar

```

Then drag and drop the file /build/libs/shard-smp-{version}.jar into your server's plugins folder.

## Usage

To run a server in IntelliJ Idea, run:
```sh

./gradlew  runServer

```
